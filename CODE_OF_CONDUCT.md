# Code of Conduct

## Our Pledge

We want this project to be a respectful and constructive place for contributors and users.

## Expected Behavior

- Be respectful in issues, pull requests, and discussions.
- Focus on facts, evidence, and clear reasoning.
- Accept feedback professionally.
- Help keep documentation understandable for non-technical users.

## Unacceptable Behavior

- Personal attacks or harassment
- Threatening behavior
- Sharing private information without permission
- Disruptive or bad-faith conduct

## Enforcement

Report problems to the maintainer through the repository owner profile on GitHub. Reports will be handled privately.
